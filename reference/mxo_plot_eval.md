# Plot the heuristic-evaluation curve along a game

Plot the heuristic-evaluation curve along a game

## Usage

``` r
mxo_plot_eval(record, ...)
```

## Arguments

- record:

  An `mxo_game_record` from
  [`mxo_self_play()`](https://r-heller.github.io/multixoR/reference/mxo_self_play.md).

- ...:

  Unused.

## Value

A `ggplot` object.
